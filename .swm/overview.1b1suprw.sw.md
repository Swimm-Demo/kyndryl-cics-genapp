### Commercial Policy (LGTESTP4)

Commercial Policy (LGTESTP4) is a program designed to handle commercial policy transactions. It provides a menu for users to insert, query, and delete policy information stored in an IBM Db2 database. The program initializes various fields, displays a main menu, handles user input, and performs actions based on the user's selection. These actions include querying policy details, inserting new policies, and deleting existing policies.
